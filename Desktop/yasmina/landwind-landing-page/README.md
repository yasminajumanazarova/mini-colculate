# Tailwind Landing Page (Play CDN)

Bu project — **HTML + Tailwind CSS (Play CDN)** yordamida tayyorlangan landing page.  
Minimal, tez ishlaydigan va front-end practice uchun qulay layout.

## ✨ Features

- Responsive layout (desktop / tablet / mobile) (feature)
- Tailwind CSS **Play CDN** orqali ulangan
- Section’lar: Navbar, Hero, Logos, Features, Stats, Testimonials, Pricing, FAQ, CTA, Footer
- Toza HTML strukturasi (semantic sectionlar)

## 🧩 Tech Stack

- HTML5
- Tailwind CSS (Play CDN)
